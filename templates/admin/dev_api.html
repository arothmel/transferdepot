Developer API Door

The public REST interface is still live for automation-loving friends. A few quick reminders:

Health check

curl {{ base_url }}/api/v1/admin/healthz

Upload

curl -F "file=@large.bin" {{ base_url }}/api/v1/upload/{{ example_group }}

List files

curl {{ base_url }}/api/v1/files/{{ example_group }}

Download

curl -OJ {{ base_url }}/api/v1/files/{{ example_group }}/{{ example_filename }}

Need raw status data? Poll the heartbeat JSON written to {{ base_url }}/{{ example_group }}/status or call the admin telemetry endpoint once bwatch is available.
